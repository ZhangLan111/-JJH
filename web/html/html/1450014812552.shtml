我校隆重举行产学研合作暨教育基金会成立大会

日期：2016-01-06   供稿单位：基金会   

2012年12月1日上午，我校产学研合作暨教育基金会成立大会在第三国际会议厅隆重举行。出席就坐大会主席台的领导和嘉宾有：中国民族贸易促进会执行会长、中国产学研合作促进会品牌战略联盟高级顾问刘延宁，工业和信息化部电信管理局正司局级巡视员张新生，重庆市经济和信息化委员会副主任杨孝荣，重庆市科学技术委员会副主任徐青，中国科学院微电子研究所所长、重庆国际半导体学院理事会理事长叶甜春，中科院沈阳自动化研究所所长于海斌，中国电子科技集团第三十八研究所党委书记陈学军，重庆市教育委员会总会计师邓睿、重庆市民间组织管理局副局长余东海，中国产学研合作促进会副会长兼秘书长王建华，校党委书记陈流汀，校长李银国。

来自大唐电信科技产业控股有限公司、华为技术有限公司、中兴通讯股份有限公司、联想移动通信科技有限公司、中国四联仪器仪表集团有限公司，通鼎集团有限公司、重庆交通科研设计院有限公司、重庆慧居智能电子有限公司、重庆渝臻投资有限公司、重庆三峡银行股份有限公司、中国长城计算机深圳股份有限公司、迈普通信技术股份有限公司、中国科学院声学研究所、武汉邮电科学研究院、工业和信息化部电信研究院西部分院等学校董事会董事及董事代表和捐赠与合作单位代表、重庆市级部门相关处室负责人出席了大会。学校副校长颜虹、林金朝、杜惠平，全校副处级以上干部、教授及正高职称人员、教学部/实验中心负责人、校级及以上科研机构负责人、各单位产学研工作秘书、教代会/职代会代表、研究生会/学生会代表350余人参会。陈流汀主持大会。

大会分两个阶段举行，第一阶段为产学研合作大会开幕式暨教育基金会揭牌仪式。

市科委副主任徐青在开幕式上致辞时强调，开展产学研合作，服务地方经济社会发展，是高等学校的重要职责和任务，是促进高校自主创新、培养创新人才的重大举措，是实现高校健康、持续、协调发展的内在要求，对于高校培养高素质创新人才、推动科技创新、促进科技成果转化具有重大而深远的意义。重庆邮电大学在重庆高校中率先召开产学研合作工作大会，充分体现了学校党政对推进产学研合作的高度重视，希望学校师生以此次会议召开为契机，在校党政的正确领导下，开拓创新，锐意进取，不断取得新的成绩，努力开创产学研合作工作的新局面。

在教育基金会成立揭牌仪式上，市民政局副局长余东海宣读了“重庆市民政局关于重庆邮电大学教育基金会设立登记的批复”文件，决定准予重庆邮电大学教育基金会设立登记，该基金会属非公募基金会，业务主管单位为重庆市教育委员会。副校长林金朝介绍了学校基金会筹备组建和相关情况，在与会人员的共同见证下，杨孝荣、邓睿、余东海、李银国共同为基金会揭牌。

揭牌仪式后还举行了捐赠与合作协议签字仪式。副校长颜虹介绍了捐赠与合作协议的主要内容，我校近期与21家单位达成了合作协议，涉及包括战略合作、科研项目合作、奖学金助学金捐赠、学生科技创新资助、科研平台共建等方面，经费共计2260万元。李银国代表学校与相关单位签订了合作协议。

开幕式最后，王建华代表中国产学研合作促进会致辞时指出，重庆邮电大学党政领导高度重视产学研合作，成立了专门的行政职能部门管理和促进产学研合作工作，搭建了学校董事会产学研合作平台，积极探索产学研合作模式，通过产学研合作在科技创新、人才培养、成果转化等方面取得了突出成果，并荣获了2012年“中国产学研合作促进奖”。他希望，重庆邮电大学继续贯彻党的十八大和全国科技创新大会精神，及时总结所取得的成功经验和做法，加快学校产学研协同创新的步伐，为学校进一步发展作出新的贡献。

大会第二阶段，李银国作了《坚持开放办学，加强产学研合作，全面提升办学水平和服务社会能力》的主题报告。他指出本次大会是学校全面落实“十二五”事业发展规划，大力实施开放办学战略，积极筹备博士学位授予单位立项验收，全力推进高水平教学研究型大学建设的关键时期召开的一次重要会议。报告从产学研合作管理体系、发展模式及取得的成效等方面对学校近五年产学研合作工作进行了回顾和总结，全面分析了学校产学研工作面临的新形势和挑战，进一步明确了今后产学研合作必须坚持“拓展渠道、整合资源、完善管理、共谋发展”的工作方针，提出了全面启动和实施 “2356产学研合作工程”的任务，并要求切实落实相应的“六类保障措施”。

主题报告结束后，通信学院陈前斌院长、计算机学院王国胤院长、自动化学院王平院长、软件学院钱鹰院长、法学院熊志海院长分别就其学院产学研合作开展情况进行了大会交流发言。

陈流汀最后讲话强调，产学研合作既是经济社会发展的需要，更是学校进一步发展的需要，涉及学校办学体制及运行机制的创新，是关乎学校今后发展和前途命运的大事，是需要举全校之力，需要大家特别是相关部门和单位共同努力、共同参与、共同推进的重点工程。
