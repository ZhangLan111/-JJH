重庆邮电大学教育基金会第一届理事会第五次会议召开

日期：2016-01-06   供稿单位：基金会   

2014年12月30日上午，重庆邮电大学教育基金会第一届理事会第五次会议在香樟园二会议室召开。基金会名誉理事长、校长李银国，基金会副理事长、校党委副书记游敏惠，基金会副理事长、副校长吕翊，基金会副理事长、副校长刘宴兵，重庆巴蜀电信发展有限公司总经理廖传利等19位理事、监事出席了会议。会议由副理事长、副校长林金朝主持。

会上，李银国宣读了市民政局关于校基金会法人变更的批复，同意我会法定代表人变更为林金朝。与会人员听取了《2014年度基金会工作报告》，包括基金会的年检情况、资金筹集及发放工作情况、重大捐赠项目、重大业务活动等，并通报了2014年度基金会收支情况。

游敏惠、吕翊、刘宴兵、廖传利对基金会的宣传工作和未来发展提出了许多宝贵的意见。各位理事、监事积极建言献策，就基金会2015年工作计划及重点工作进行了深入探讨。

最后，名誉理事长李银国作了总结讲话，他充分肯定了一年来教育基金会的工作，对基金会2015年工作提出了具体的要求和建议；他希望各位理事、监事集思广益，学校相关单位团结协作，大力支持基金会的工作；同时，基金会要不断拓宽基金的筹资渠道，切实提高基金的使用效果，为学校的建设和发展作出更大的贡献。

学校教育基金会第一届理事会第五次会议召开（记者 王策 摄）
